# TailNG – Angular 21

Nx + Angular 21 + Tailwind starter for TailNG.

## Included
- apps/docs – demo & docs site
- apps/playground – sandbox
- libs/ui – Button, Input, Card
- libs/icons – Bootstrap Icons wrapper
- libs/theme – Tailwind preset + plugin
- libs/cdk – utilities

## Run
```bash
yarn
yarn docs
```
