ListBox Demo (Vanilla CSS)

CDK-backed multi-select listbox with active-descendant focus, range selection, and typeahead.

Keyboard Controls

  • Arrow Up/Down move active option
  • Home/End jump to first/last option
  • Space toggle active selection
  • Enter select only active option
  • Shift + Arrow extend contiguous range
  • Ctrl + A select all enabled
  • Tab leave listbox naturally
  • Type letters typeahead jump
