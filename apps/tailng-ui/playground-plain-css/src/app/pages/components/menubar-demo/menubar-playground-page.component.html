Keyboard Contract

  • ArrowLeft / ArrowRight moves across top-level items.
  • ArrowDown opens an owned menu and focuses the first enabled item.
  • Escape closes the menu and restores focus to the owning top-level item.
  • Home / End jump to the first or last enabled top-level item.
