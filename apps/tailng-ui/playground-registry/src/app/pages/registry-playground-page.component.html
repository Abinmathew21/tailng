TailNG Registry Playground

CLI Integration Test App

Use this app as a clean target for `tailng add ...` commands from the local monorepo CLI.

Step 1: Run TailNG CLI

pnpm tailng -- add button --cwd apps/tailng-ui/playground-registry

Start with --dry-run if you want to preview file writes first. 

pnpm tailng -- add button --cwd apps/tailng-ui/playground-registry --dry-run

Step 2: Review Generated Files

  • src/app/tailng-ui/button/tng-button-primitive.ts
  • src/app/tailng-ui/button/tng-button.ts
  • src/app/tailng-ui/button/tng-button.html
  • src/app/tailng-ui/button/tng-button.css
  • src/app/tailng-ui/button/index.ts

Step 3: Wire It In

Import from src/app/tailng-ui/button and render <tng-button> in any page to validate end-to-end behavior.
