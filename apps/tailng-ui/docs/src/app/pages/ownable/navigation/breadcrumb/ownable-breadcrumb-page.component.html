Breadcrumb

The ownable Breadcrumb install gives your app a local wrapper you can tune for separator styling, collapsed trails, and product-specific hierarchy copy while keeping the TailNG semantics intact.

Why own Breadcrumb locally

Breadcrumbs usually absorb routing conventions, truncation rules, and condensed mobile layouts that vary by product. Local ownership keeps those decisions close to the app.

  • Adjust separators, icons, and current-item treatments without waiting on a package release.
  • Align collapsed-path behavior with your route and IA conventions.
  • Version breadcrumb presentation beside the pages that depend on it.

What the install includes

The registry-backed breadcrumb install writes a local wrapper component, template, styles, and export barrel under your app so navigation polish can evolve with the surrounding docs or product shell.
