API reference

Headless stepper is a structural primitive. It marks the root with a stable slot selector and leaves step rows, states, and any interactivity to the owning markup.

tngStepper

Attach the directive to the container that owns your ordered progress content. The primitive does not add step-item directives, outputs, or keyboard behavior.

Surface Value Purpose
Selector [tngStepper] Attaches the primitive to your chosen host element.
Export alias tngStepper Lets templates reference the directive instance if needed.
Host hook data-slot="stepper" Stable selector for styling and queries.
Built-in state hooks None Current, complete, and error state semantics are fully owner-authored.

Owner responsibilities

  • Choose the correct semantic container, usually an ordered list for linear flows.
  • Author step rows, labels, icons, connectors, and state tokens in your own markup.
  • Expose the active step with aria-current="step" when the flow has a current position.
  • Add any keyboard or link behavior yourself if step rows are navigable.
