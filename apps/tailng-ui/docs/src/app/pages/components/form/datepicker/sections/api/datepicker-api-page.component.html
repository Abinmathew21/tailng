API reference

The default docs surface is the tng-datepicker wrapper. When the wrapper is too opinionated for a product flow, use createDatepickerController(...) and render the same slots yourself.

tng-datepicker (component)

Inputs

Property Type Default Details
adapter TngDateAdapter<TDate> | undefined Default Date adapter Controls parsing/formatting for the input field and overlay labels.
defaultValue TngDateSelectionInput<TDate> | undefined undefined Uncontrolled initial selection.
value TngDateSelectionInput<TDate> | undefined undefined Controlled selected value.
minDate / maxDate TngDateInputValue<TDate> | undefined undefined Disables out-of-range days, months, and years.
allowManualInput boolean true Allows direct keyboard editing in the input field.
locale string Angular LOCALE_ID Drives month names, weekday labels, and adapter formatting defaults.
placement 'auto' | 'bottom' | 'top' 'auto' Auto-flips the overlay when there is not enough room below the trigger.
placeholder string 'MM-DD-YYYY' Input placeholder only. It does not change parsing rules.
today TngDateInputValue<TDate> | undefined Adapter today() Overrides the date flagged as today in the overlay grid.
weekStartsOn 0 | 1 | 2 | 3 | 4 | 5 | 6 0 Controls weekday order and grid alignment.
disabled / readonly / invalid boolean false Drives wrapper state styling and interaction rules.

Outputs

Event Payload Details
valueChange TngDateValue<TDate> Emits after click, keyboard commit, or valid manual input commit.
openChange boolean Emits when the popup opens or closes.
closed TngDatepickerCloseReason Reports escape, outside, programmatic, or select.
activeDateChange TDate Emits as keyboard focus moves through the calendar.
viewChange 'day' | 'month' | 'year' Tracks the material-style year → month → day drill-down flow.
monthChange TDate Emits when paging changes the visible month block.
yearChange number Emits when the visible year page anchor changes.

createDatepickerController(...) (headless)

Method Purpose
getOutputs() Returns the slot/state payload used to render day, month, year, trigger, and overlay UI.
showYearsPanel(), showMonthsPanel(), showDaysPanel() Lets the implementation page drive the visible panel explicitly.
pageForward() / pageBackward() Moves the visible month/year block in the currently active view.
setInputText(...) / commitInputText() Supports manual input editing with the same adapter/bounds validation as the wrapper.
registerTrigger(...) / registerOverlay(...) Hooks focus restoration, outside dismissal, and overlay runtime behavior into custom UI.
