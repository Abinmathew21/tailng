API reference

MultiAutocomplete exposes a headless primitive stack and a component wrapper with value-array mapping.

tngMultiAutocomplete + parts (primitive)

Input / model Type Notes
open boolean Overlay open state.
value readonly T[] Selected option values (multi-select model).
query string Current trigger text used for filtering.
disabled, loading, invalid boolean State hooks reflected on attributes and ARIA.
openChange, valueChange, queryChange Output events Emitted on interaction commits and query updates.
Methods add/remove/toggle/clear Imperative selection helpers on primitive export.

tng-multi-autocomplete (styled component)

API Type Details
options readonly O[] Option source for rendering and filtering.
getOptionValue (option: O) => V Maps options to committed value model.
getOptionLabel (option: O) => string Maps options to display/filter label.
isOptionDisabled (option: O) => boolean Disables specific options.
Forwarded primitive model open, value, query, openChange, valueChange, queryChange Exposed through host directive wiring.
