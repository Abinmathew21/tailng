Form Field

<tng-form-field> is the shell component for a projected input[tngInput] plus optional tngPrefix/tngSuffix content.

Use it when the field needs icons, fixed suffix text, or trailing actions. For the plain case with no projected adornments, use <tng-input> instead.

What form field does

  • Owns the shell: sizing, border, focus ring, and inline layout.
  • Projects one real input: the native control still comes from input[tngInput].
  • Supports adornments: leading and trailing content stay outside the native control while sharing the same field chrome.

Installation

Import the shell component from components and the projected-control primitives from primitives.

Relationship to input component

<tng-input> is the simple component for a plain field. Reach for <tng-form-field> when the field needs projected prefix/suffix content or any custom trailing action.

Accessibility guidance

  • Mark decorative prefix/suffix content as aria-hidden="true".
  • Give the projected input a real accessible name with label or ARIA.
  • Trailing buttons should keep their own explicit accessible label.
