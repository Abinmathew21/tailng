Styling contract

Style against switch state hooks and semantic tokens. Do not depend on internal DOM beyond the documented attributes.

CSS contract table

Selector Applied on Purpose
[data-state="checked"] Primitive button Checked track styling (e.g. brand color).
[data-state="unchecked"] Primitive button Unchecked track styling (e.g. muted background).
[data-disabled] Primitive button + wrapper root Muted visuals and not-allowed cursor.
[data-slot="switch"] Primitive button Theme contract anchor for track element.
