Behavior baseline

  • Trigger exposes role="combobox" with listbox state attributes.
  • ArrowUp/ArrowDown opens and traverses options.
  • Enter commits the active option and closes the overlay.
  • Styling hooks are available via slots like select-trigger, select-overlay, and select-option.
