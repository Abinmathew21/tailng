Styling contract

Style against checkbox state hooks and semantic tokens. Do not depend on internal DOM beyond the documented attributes.

CSS contract table

Selector Applied on Purpose
[data-state="checked"] Primitive input Checked state styling.
[data-state="mixed"] Primitive input Tri-state mixed appearance.
[data-disabled] Primitive input + wrapper root Muted visuals and not-allowed cursor behavior.
[data-readonly] Primitive input + wrapper root Readonly visual treatment separate from disabled.
[data-invalid] Primitive input + wrapper root Error emphasis and validation styling.
[data-focused], [data-focus-visible] Primitive input Focus and keyboard-focus visual differentiation.
