Styling contract

Select styling should target public slot and state attributes, so custom skins remain stable across internal implementation updates.

CSS contract table

Selector Applied on Purpose
[data-slot="select"] Select host Local token overrides and state container.
[data-slot="select-trigger"] Trigger control Trigger shape, focus ring, and typography.
[data-slot="select-overlay"] Portaled overlay Surface, border, and elevation styling.
[data-slot="select-option"][data-active] Option host Keyboard-active option highlight.
[data-slot="select-option"][data-selected] Option host Committed value treatment.
