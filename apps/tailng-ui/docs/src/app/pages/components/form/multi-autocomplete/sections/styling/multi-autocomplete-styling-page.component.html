Styling contract

Style against documented data-slot and state attributes instead of internal DOM assumptions.

CSS contract table

Selector Applied on Purpose
[data-slot='multi-autocomplete'] Root host Root layout, chip wrapping, and state hooks.
[data-slot='multi-autocomplete-trigger'] Input trigger Text input surface, typography, and placeholder styles.
[data-slot='multi-autocomplete-chip'] Selected chip Selected item token styling and remove action target.
[data-slot='multi-autocomplete-overlay'] Overlay panel Panel container for borders, shadows, spacing.
[data-slot='multi-autocomplete-option'][data-active] List option Keyboard active styling.
[data-slot='multi-autocomplete-option'][data-selected] List option Selected option styling.
