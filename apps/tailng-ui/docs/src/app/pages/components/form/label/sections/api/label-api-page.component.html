API reference

label[tngLabel] exposes a compact headless contract while <tng-label> adds the default wrapper markup and required marker.

Primitive: label[tngLabel]

Input Type Default Notes
disabled boolean false Adds data-disabled hook for styling.
required boolean false Sets data-required; keep aria-required on the control.

Component: <tng-label>

Input Type Default Notes
forId string '' Maps to native for when trimmed value is non-empty.
required boolean false Shows visual asterisk and forwards data-required.
disabled boolean false Forwards data-disabled state hook to the inner label.

State attributes

  • data-slot="label" is always present.
  • data-required appears when required is enabled.
  • data-disabled appears when disabled is enabled.
