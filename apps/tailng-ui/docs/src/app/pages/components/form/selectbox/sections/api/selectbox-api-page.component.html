API reference

SelectBox exposes composable primitives and a styled wrapper. Choose primitives for full markup ownership, or use <tng-select> for faster integration.

Primitive directives

Directive API Type Notes
tngSelect open, openChange boolean Controlled overlay visibility.
tngSelect value, valueChange T | null Single selected value model.
tngSelect disabled, invalid, loading boolean Global interaction and state flags.
tngSelectListbox orientation, direction, loop 'vertical' | 'horizontal', 'ltr' | 'rtl', boolean Keyboard traversal behavior for options.
tngSelectOption tngValue, disabled T, boolean Option identity and disabled state.

Styled wrapper <tng-select>

Wrapper inputs include options, getOptionValue, getOptionLabel, isOptionDisabled, and all forwarded primitive models like value/valueChange.
