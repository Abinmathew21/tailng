Overview

TailNG Input provides a semantic form field container with labels, hints, validation state, and adornment slots while keeping behavior accessible by default.

What it covers

  • Label + helper text layout through tng-input.
  • Control directive for input and textarea via tngInput.
  • Leading/trailing adornments using tngInputLeading and tngInputTrailing.
  • Accessible invalid/required wiring through ARIA-aware bindings.

When to use

Use Input for all single-line and multiline text collection where consistent spacing, state styles, and accessibility should be standardized across the product.
