API reference

ListBox exposes headless directives for host and option behavior. Use this contract when building custom layouts.

tngListbox + tngOption

Directive API Type Notes
tngListbox multiple boolean Switches single vs multiple selection behavior.
tngListbox orientation 'vertical' | 'horizontal' Arrow-key axis for navigation.
tngListbox direction 'ltr' | 'rtl' Horizontal key mapping direction.
tngListbox disabled, loop, typeahead boolean Global interactivity and keyboard traversal behavior.
tngListbox value, valueChange T | readonly T[] | null Selection model and output updates.
tngOption tngValue T Option value committed to the listbox model.
tngOption disabled boolean Skips option from keyboard and selection interactions.

Wrapper status

A dedicated tng-listbox component wrapper is not yet part of @tailng-ui/components. For wrapped combobox experiences, use tng-select or tng-multiselect.
