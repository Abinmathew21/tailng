Toggle overview

TailNG toggle provides pressed-button semantics through a headless directive (tngToggle) and a styled wrapper (tng-toggle).

Imports

Use primitives for full ownership, or use the styled component for fast adoption.

Usage patterns
B B I I Disabled
grid grid list list disabled

Accessibility baseline

  • aria-pressed reflects the active state for assistive technologies.
  • Native button semantics preserve keyboard support for Enter, Space, and Tab flow.
  • State hooks are exposed via data-state and focus data attributes.
