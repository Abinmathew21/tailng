Textarea overview

Textarea in TailNG is now input-first. Use textarea[tngInput] for headless control, then layer <tng-textarea> when you want the default component shell.

Behavior baseline

  • Uses native <textarea> behavior for input, selection, and IME composition.
  • rows is normalized to at least 1.
  • resize accepts vertical, horizontal, both, or none.
  • Legacy textarea[tngTextarea] remains as a compatibility alias while textarea[tngInput] is the preferred primitive path.
