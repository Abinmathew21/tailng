Styling contract

Datepicker styling is slot/state driven. The wrapper and the headless demos share the same contract, so custom themes should target the public data attributes instead of internal DOM structure.

Slot selectors

Selector Purpose
[data-slot="datepicker"] Host contract root for field-level layout tokens and shared semantic colors.
[data-slot="datepicker-field"] Field wrapper containing the input shell and the overlay anchor.
[data-slot="datepicker-input-shell"] Combined input/trigger chrome. Use this for border, focus ring, and invalid treatment.
[data-slot="datepicker-input"] Editable text input surface.
[data-slot="datepicker-trigger"] Calendar button suffix inside the input shell.
[data-slot="datepicker-overlay"] Popup panel. This is important because the overlay is portaled to body.
[data-slot="datepicker-period-button"] Centered month-year button used to drill into year and month panels.
[data-slot="datepicker-grid"] Month/year grid container.
[data-slot="datepicker-cell"] Day cell slot for the calendar grid.
[data-slot="datepicker-month"] Month picker cell slot.
[data-slot="datepicker-year"] Year picker cell slot.

State hooks

State Applied on Use case
[data-open] Host, trigger, input shell, overlay Open-state affordances and trigger emphasis.
[data-invalid] Input shell Manual input parse/bounds failure treatment.
[data-disabled] Host, day/month/year cells Disabled field or out-of-range date visuals.
[data-selected] Day/month/year cells Committed selection state.
[data-focus-visible] Day/month/year cells Keyboard focus treatment independent of selected state.
[data-active] Day/month/year cells Current roving target when focus and selection differ.
[data-in-month] Day cells Distinguishes outside days from the visible month.

Common custom properties

Token Typical usage
--tng-datepicker-radius Input shell and overlay radius.
--tng-datepicker-field-height Combined input/trigger height.
--tng-datepicker-overlay-padding Outer padding inside the popup panel.
--tng-datepicker-grid-gap Responsive spacing between day, month, and year cells.
--tng-datepicker-day-cell-size Base day-cell height.
--tng-datepicker-picker-cell-size Base month/year-cell height.
--tng-datepicker-brand Selected and focus-visible accents.
--tng-datepicker-border / --tng-datepicker-surface Field chrome and overlay surface styling.

Compact override example

These tokens are already tuned for the default compact docs examples. Override them on a wrapper scope when a denser or roomier product surface needs a different rhythm.
