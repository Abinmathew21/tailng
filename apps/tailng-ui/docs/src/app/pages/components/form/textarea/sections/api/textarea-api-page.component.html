API reference

Textarea supports both the preferred input-first primitive (textarea[tngInput]) and the styled wrapper (<tng-textarea>).

Primitive: textarea[tngInput]

Input Type Default Notes
rows number 3 Normalized to a minimum of 1.
resize 'vertical' | 'horizontal' | 'both' | 'none' 'vertical' Applied to both style.resize and data-resize.
disabled, readonly, required boolean false Forwarded as native textarea attributes.
ariaInvalid, ariaRequired boolean | null null ARIA pass-through follows nullable coercion rules.

Component: <tng-textarea>

Input / Output Type Default Notes
value string | null null Controlled value for the inner native textarea.
valueChange EventEmitter<string> — Emits on input events when the value changes.
placeholder string | null null Maps to native placeholder.
rows, resize number, resize enum 3, vertical Forwarded to primitive-backed textarea contract.

Compatibility alias

textarea[tngTextarea] is still exported for backward compatibility and delegates to the same underlying behavior as textarea[tngInput].
