API reference

Switch exposes a primitive directive for headless behavior and a styled wrapper with label, hidden input, and arrow-key support.

tngSwitch (directive)

Attach on button. Sets role="switch" automatically.

Input Type Default Notes
checked boolean | '' | string false Current on/off state.
disabled boolean | '' | string false Blocks interaction and applies disabled attributes.
required boolean | '' | string false Sets aria-required.
ariaLabel string | null null Accessible name when no visible label is present.
Output Attribute When set Purpose
role="switch" Always ARIA switch role.
aria-checked Always "true" or "false".
data-state Always checked or unchecked.
data-disabled When disabled Styling hook for disabled state.
data-slot="switch" Always Theme contract anchor.

tng-switch (styled component)

Wrapper with track/thumb, hidden native input for form submission, and arrow-key toggling.

API Type Details
Inputs checked, disabled, required, name, value, ariaLabel Forwarded to primitive and hidden input.
Outputs checkedChange Emits on toggle when not disabled.
Keyboard Space, ArrowLeft, ArrowRight Space toggles; ArrowLeft unchecks; ArrowRight checks.
Content ng-content Projected as visible label text.
Icon slots [onIcon], [offIcon] Optional projected icons shown inside the thumb.
