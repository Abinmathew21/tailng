Button Toggle overview

Button Toggle composes selectable buttons in single or multiple mode for toolbars, filters, and compact state controls.

Imports

Use primitives for full ownership, or use the styled components for fast adoption.

Usage patterns
Compact Comfortable Spacious
Accessibility Signals DX

Accessibility baseline

  • Single mode uses radiogroup/radio semantics; multiple mode uses pressed buttons.
  • Arrow keys move focus across enabled items; Home and End jump to first and last.
  • State hooks are exposed through data-selected, data-state, and focus attributes.
