Styling contract

MultiSelect styling should target public slots and state attributes for predictable custom skins.

CSS contract table

Selector Applied on Purpose
[data-slot="multi-select"] Root host State container and token overrides.
[data-slot="select-trigger"] Trigger control Trigger surface and focus ring.
[data-slot="select-overlay"] Portaled overlay Panel border, radius, and elevation.
[data-slot="multi-select-listbox"] Listbox host Option stack spacing and scrolling behavior.
[data-slot="multi-select-option"][data-active] Option host Keyboard active highlight.
[data-slot="multi-select-option"][data-selected] Option host Committed selection styling.
