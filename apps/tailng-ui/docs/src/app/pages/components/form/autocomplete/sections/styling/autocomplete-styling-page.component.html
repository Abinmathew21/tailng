Styling contract

Style against documented data-slot and state attributes instead of internal DOM assumptions.

CSS contract table

Selector Applied on Purpose
[data-slot='autocomplete'] Root host Root layout and state hooks.
[data-slot='autocomplete-trigger'] Input trigger Input surface, typography, focus styles.
[data-slot='autocomplete-overlay'] Overlay panel Panel container for borders, shadows, spacing.
[data-slot='autocomplete-option'][data-active] List option Keyboard active styling.
[data-slot='autocomplete-option'][aria-selected='true'] List option Selected value styling.
