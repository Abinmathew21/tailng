Styling

Input styling is token-driven. Override semantic variables globally or at component scope.

Common tokens

  • --tng-semantic-background-surface for control background.
  • --tng-semantic-border-subtle for default border.
  • --tng-semantic-focus-ring for focus-visible state.
  • --tng-semantic-accent-brand for active/highlighted accents.

Scoped override example

.account-form {
  --tng-semantic-accent-brand: #0f766e;
  --tng-semantic-focus-ring: rgba(15, 118, 110, 0.35);
  --tng-semantic-border-subtle: #99f6e4;
}
