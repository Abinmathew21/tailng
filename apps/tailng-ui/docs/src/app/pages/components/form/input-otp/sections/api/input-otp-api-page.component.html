API reference

Input OTP ships as a primitive directive for state hooks and a wrapper component for complete slot behavior, paste handling, and form integration.

tngInputOtp primitive

Directive API Type Notes
tngInputOtp length number (1..12) Total OTP slot count exposed through host state hooks.
tngInputOtp value string Combined OTP value used to derive data-empty/partial/complete.
tngInputOtp activeIndex number | null Marks the active slot through data-active.
tngInputOtp disabled, readonly, required, invalid, focused, focusVisible boolean Maps interaction and validation state into data attributes for styling.

<tng-input-otp> wrapper

Wrapper component handles slot rendering, keyboard traversal, paste distribution, hidden input submission, and emits valueChange + complete.

Angular forms integration

The wrapper implements ControlValueAccessor and works with formControl, formControlName, and ngModel.
