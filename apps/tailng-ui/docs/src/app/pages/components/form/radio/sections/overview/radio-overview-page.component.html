Radio overview

TailNG radio supports native single-selection behavior with a headless primitive (tngRadio) and a styled wrapper (tng-radio).

Imports

Use primitives for full ownership, or wrappers for a themed default shell.

  • Native radio semantics are preserved for screen reader compatibility.
  • Keyboard arrow navigation is inherited from native grouped radio behavior.
