Styling contract

Button Toggle styling should target state attributes on the group and item slots.

Selector Purpose
[data-slot='button-toggle-group'] Group layout and container spacing.
[data-slot='button-toggle'][data-selected='true'] Pressed/selected visual style.
[data-slot='button-toggle'][data-focused='true'] Focus ring and keyboard emphasis.
[data-slot='button-toggle'][data-disabled='true'] Disabled visual treatment.
