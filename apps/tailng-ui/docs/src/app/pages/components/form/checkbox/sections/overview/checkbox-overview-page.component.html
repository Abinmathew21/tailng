Checkbox overview

TailNG checkbox supports binary and tri-state selection with a headless primitive (tngCheckbox) and a styled wrapper (tng-checkbox).

Imports

Use primitives for full ownership, or use the styled component for fast adoption.

Usage patterns
Email notifications Weekly digest (mixed) Readonly legal acknowledgement
Email notifications Weekly digest (mixed) Readonly legal acknowledgement

Accessibility baseline

  • Tri-state uses aria-checked="mixed" automatically.
  • Readonly stays focusable, but toggling is blocked for keyboard and pointer.
  • State hooks are exposed via data-* attributes for styling.
