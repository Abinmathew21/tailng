Label overview

Label wires accessible names to controls and keeps click-to-focus behavior native. Use label[tngLabel] for headless markup or <tng-label> for a themed wrapper.

Accessibility baseline

  • Associate labels with controls via explicit for/id or wrapping markup.
  • Keep required semantics on the control with aria-required.
  • Avoid interactive children inside the label text region.
