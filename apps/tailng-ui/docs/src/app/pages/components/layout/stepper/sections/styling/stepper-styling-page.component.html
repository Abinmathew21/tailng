Styling contract

Style against explicit data hooks from the stepper host and each step item. Keep state tokens centralized so current/completed/error remain visually consistent across flows.

State hooks

Hook Where Use
data-slot="stepper" Root Container spacing, orientation layouts, connector styling.
data-state="current" Step item Active step emphasis and readable focus baseline.
data-state="completed" Step item Past-step confirmation colors and icon treatment.
data-state="upcoming" Step item Muted future-step presentation.
data-state="error" Step item Validation/processing failure styling.
