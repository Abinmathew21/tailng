Stepper overview

Stepper wraps the headless root with a baseline shell and an accessible label, while your app still owns the projected step rows, state semantics, and any interactive behavior inside.

Imports

Import the wrapper when you want the default shell styling and ariaLabel input.

Accessibility baseline

  • Current step should expose aria-current="step".
  • If step rows are clickable, use real links or buttons in document order.
  • Optional, error, and completed states should be announced in visible or accessible text.
