API reference

Stepper currently provides a structural primitive directive and a styled wrapper component. Behavioral navigation APIs are planned for the next iteration.

tngStepper (directive)

Attach on a container element (ol, section, etc.) to expose stepper slot semantics. The rows inside are fully owner-authored.

Surface Value Purpose
Host attribute data-slot="stepper" Stable styling and query hook.
Selector [tngStepper] Attach stepper semantics without imposing markup.
Export alias tngStepper Reference directive instance from templates if needed.
Built-in item hooks None Any data-state, aria-current, or classes on step rows are app-authored.

tng-stepper (component)

Wrapper that applies a baseline shell and forwards ariaLabel to the stepper host. It does not generate step rows or manage current, completed, or error states for you.

Input Type Default Notes
ariaLabel string 'Stepper' Sets accessible label on the underlying stepper host.
