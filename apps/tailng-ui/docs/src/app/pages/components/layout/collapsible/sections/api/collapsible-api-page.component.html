API reference

Collapsible currently provides a structural primitive directive and a styled wrapper component. Behavioral navigation APIs are planned for the next iteration.

tngCollapsible (directive)

Attach on a container element (ol, section, etc.) to expose collapsible slot semantics.

Surface Value Purpose
Host attribute data-slot="collapsible" Stable styling and query hook.
Selector [tngCollapsible] Attach collapsible semantics without imposing markup.
Export alias tngCollapsible Reference directive instance from templates if needed.

tng-collapsible (component)

Wrapper that applies a baseline shell and forwards ariaLabel to the collapsible host.

Input Type Default Notes
ariaLabel string 'Collapsible' Sets accessible label on the underlying collapsible host.
