Accordion overview

Accordion exposes headless expand/collapse behavior with roving keyboard focus and accessible trigger/panel wiring. Choose primitives for full control or wrappers for faster composition.

Governance ⌄ Define release policy, reviewer gates, and rollback ownership. Delivery ⌄ Ship documentation and examples in the same milestone cadence.

Accessibility baseline

  • Triggers expose aria-expanded, aria-controls, and disabled semantics.
  • Panels expose role="region" with aria-labelledby back to the trigger.
  • Arrow keys, Home/End, Enter, and Space follow the documented keyboard contract.
