API reference

Skeleton exposes simple decorative loading controls: animation, rounding, and wrapper sizing.

tngSkeleton primitive

Attach to any block element to expose skeleton state attributes and semantics.

Input Type Default Notes
animated boolean true Toggles shimmer animation hook via data-animated.
rounded boolean true Toggles corner radius hook via data-rounded.

tng-skeleton wrapper

Wrapper component reuses primitive semantics and adds width and height inputs.
