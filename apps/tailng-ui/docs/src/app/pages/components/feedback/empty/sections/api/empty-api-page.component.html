API reference

Empty is a structural surface. Use primitives for full markup control or wrapper components for an aligned default layout contract.

tngEmpty and part directives

Apply directives to your own markup. The primitive only adds slot hooks and does not impose interaction behavior.

Directive Selector Data slot Purpose
TngEmpty [tngEmpty] empty Root container for the empty state surface.
TngEmptyIcon [tngEmptyIcon] empty-icon Optional icon/illustration region.
TngEmptyTitle [tngEmptyTitle] empty-title Primary headline describing the state.
TngEmptyDescription [tngEmptyDescription] empty-description Supporting context text.
TngEmptyActions [tngEmptyActions] empty-actions Action row for retry, create, or navigation buttons.

tng-empty wrappers

Wrapper components mirror the primitive parts and add default presentation. The root exposes one input:

  • align: 'center' | 'start' (default 'center')
