API reference

Progress Bar exposes a minimal contract: range values, indeterminate mode, and semantic accessibility attributes.

tngProgressBar and tngProgressBarIndicator

Use primitive directives when you want full control over surrounding markup and motion.

Input Type Default Notes
min number 0 Lower bound used to clamp value.
max number 100 Upper bound used to clamp value; never falls below min.
value number 0 Current determinate progress value.
indeterminate boolean false Hides aria-valuenow/aria-valuemin/aria-valuemax.

tng-progress-bar wrapper

Wrapper component uses the same inputs and adds ariaLabel plus default indicator width calculation.
