API reference

Progress Spinner exposes a compact contract: range values, indeterminate mode, and semantic accessibility attributes.

tngProgressSpinner primitive

Use the primitive directive when you want full control over spinner markup and motion.

Input Type Default Notes
min number 0 Lower bound used to clamp value.
max number 100 Upper bound used to clamp value; never falls below min.
value number 0 Current determinate progress value.
indeterminate boolean false Hides aria-valuenow/aria-valuemin/aria-valuemax.

tng-progress-spinner wrapper

Wrapper component reuses primitive semantics and adds ariaLabel, size, and strokeWidth inputs.
