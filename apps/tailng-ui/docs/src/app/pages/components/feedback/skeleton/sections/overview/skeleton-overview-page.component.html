Skeleton overview

Skeleton provides decorative placeholders that mimic content layout during loading and reduce perceived wait.

Imports

Use primitives for full control or wrapper component for default sizing inputs.

Accessibility baseline

  • Skeleton placeholders are decorative and default to aria-hidden="true".
  • Use nearby text or live regions for actual loading status communication.
  • Disable shimmer with [animated]="false" where motion should be reduced.
