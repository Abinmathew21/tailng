API reference

Codeblock exposes primitive slot directives and a styled component API for highlight, copy, and line controls.

tngCodeBlock primitive parts

Directive Host Purpose
tngCodeBlock Root container Sets data-slot="code-block" for contract styling.
tngCodeBlockHeader Header container Title/action region slot marker.
tngCodeBlockBody Body wrapper Main body slot marker for gutter + code layout.
tngCodeBlockGutter Line-number list Line-number slot marker.
tngCodeBlockCode <code> Code content slot marker for typography + tokens.

tng-code-block component inputs

Input / output Type Notes
code string Source content rendered in the block.
language, theme, adapter string | null Highlighting adapter selection and metadata.
lineNumbers, startingLineNumber boolean | 'auto', number Line-number rendering and offset.
highlightLines, focusLines Array<number | [number, number]>, boolean Line emphasis and non-highlighted dimming.
copy, copyLabel, copiedLabel boolean | 'auto', string, string Clipboard action visibility and feedback labels.
copy, copySuccess, copyError Outputs Proxy events from the copy primitive.
renderStateChange { state, error? } Render lifecycle updates (idle, highlighting, highlighted, error).
