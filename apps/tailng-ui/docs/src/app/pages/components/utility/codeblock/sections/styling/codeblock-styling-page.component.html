Styling contract

Style codeblock through documented slots and state attributes instead of internal DOM coupling.

Slot contract table

Selector Applied on Purpose
[data-slot='header'] Header row Title, language badge, and action alignment.
[data-slot='line-numbers'] Gutter list Line-number width, contrast, and typography.
[data-slot='code'] <code> container Font family, text sizing, and wrapping behavior.
[data-slot='line'] Individual rendered line Per-line active/dim/highlighted styles.
[data-slot='copy-button'] Copy action Button surface, focus ring, and success/error feedback.

State attributes

  • data-state="idle|highlighting|highlighted|error" on root.
  • data-highlighted="true|false" and data-wrap="true|false".
  • data-has-copy and data-has-title for layout conditionals.
