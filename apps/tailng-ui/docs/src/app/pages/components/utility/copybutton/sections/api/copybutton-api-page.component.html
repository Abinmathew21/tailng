API reference

<tng-copy-button> wraps the headless tngCopy primitive with stateful labels and icon slots.

Component inputs

Input Type Default Notes
text string | null | undefined undefined Explicit text payload to copy.
from TngCopyFromTarget null Fallback source element/selector when text is omitted.
ignoreSelectors string | string[] | null null Selectors removed from copied DOM text.
appearance 'outline' | 'solid' | 'ghost' 'outline' Visual treatment for wrapper button.
size 'sm' | 'md' 'md' Controls wrapper button dimensions.
showIcon boolean true Shows or hides icon slot content.
disabled boolean false Disables copy interactions.
resetDelay number 1500 Milliseconds before state returns to idle.

Component outputs

Output Payload Description
tngCopied string Emits copied text when copy succeeds.
tngCopyError Error Emits normalized error when copy fails.

Primitive passthrough events

For full event detail (tngCopy, tngCopySuccess, tngCopyError, keyboard hotkeys, announce policies), use tngCopy directly on a host element.
