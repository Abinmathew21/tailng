API reference

<tng-avatar> wraps the headless avatar primitives and handles fallback display automatically when sources are missing or fail to load.

Component inputs

Input Type Default Notes
src string | null null Image source URL. Empty/null resolves to fallback.
fallback string | null null Fallback source text converted into initials when possible.
alt string | null 'Avatar' Alt text passed to image node when present.
shape 'circle' | 'square' 'circle' Controls avatar frame geometry.
size 'sm' | 'md' | 'lg' 'md' Controls host dimensions and fallback typography scale.

Headless primitives

Directive Usage
tngAvatar Avatar host element that exposes shape/size state attributes.
tngAvatarImage Image node bound to host slot state. Hide/show is consumer controlled.
tngAvatarFallback Fallback content slot (initials/icon) for missing or failed sources.
