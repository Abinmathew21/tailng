CopyButton overview

CopyButton combines the headless tngCopy primitive with a styled <tng-copy-button> wrapper for fast clipboard UX.

Imports

Use the primitive for full ownership, or use the component for built-in copy states.

Accessibility baseline

  • Use a native button host to preserve Enter/Space activation and tab semantics.
  • State is exposed for styling through data-state on the wrapper component.
  • Copy completion messaging is sent to an internal aria-live="polite" region for assistive feedback.
