Styling contract

Style wrapper and headless hosts through stable data attributes instead of targeting internal element structure.

Wrapper state hooks

Attribute Values Usage
data-appearance solid | outline | ghost Selects visual button style.
data-tone primary | neutral | success | danger Maps semantic color intent.
data-size sm | md | lg Controls component dimensions.
data-disabled "" when disabled Use for styling disabled affordance.

Primitive host behavior

  • On anchors, aria-disabled="true" is set when disabled.
  • Anchors without href get role="button" automatically.
  • Keyboard activation on anchor hosts is normalized for Enter and Space.
