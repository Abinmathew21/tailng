API reference

<tng-button> wraps tngPress. Use the wrapper for visual tokens, or use the primitive directly for headless control.

Wrapper inputs (<tng-button>)

Input Type Default Notes
appearance 'solid' | 'outline' | 'ghost' 'solid' Visual style variant.
tone 'primary' | 'neutral' | 'success' | 'danger' 'primary' Semantic color treatment.
size 'sm' | 'md' | 'lg' 'md' Control height and horizontal padding.
type 'button' | 'submit' | 'reset' 'button' Native button type forwarding.
disabled boolean false Disables interaction and sets disabled host state.
ariaPressed boolean | null null Pressed-state semantics for toggle behavior.
ariaExpanded boolean | null null Expanded state for disclosure/menu triggers.
ariaHasPopup TngPressAriaHasPopup | null null Popup relationship hint (menu, dialog, listbox, etc.).
ariaControls string | null null ID reference for controlled content.

Primitive inputs (tngPress)

Input Type Default Notes
type 'button' | 'submit' | 'reset' 'button' Applied when host is a native button.
disabled boolean false Blocks click/keyboard activation and adds disabled semantics.
ariaPressed, ariaExpanded boolean | null null Nullable coercion to ARIA boolean attributes.
ariaHasPopup 'menu' | 'dialog' | ... | 'true' | 'false' null Normalized allowed values only.
ariaControls string | null null Trimmed to null when empty/whitespace.

Behavior notes

  • Anchor hosts without href get role="button" and keyboard activation.
  • Disabled anchors expose aria-disabled and are removed from tab order.
  • Wrapper component emits native click events from the internal button host.
