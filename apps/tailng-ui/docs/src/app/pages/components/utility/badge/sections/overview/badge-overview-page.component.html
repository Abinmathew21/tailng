Badge overview

Badge is a lightweight overlay primitive for notification counts and status dots. Use tngBadge directly for headless control or from @tailng-ui/components for wrapper-level integration.

Accessibility baseline

  • Badge bubble is decorative by default and marked with aria-hidden="true".
  • Provide meaningful host labeling (for example aria-label="Inbox notifications").
  • Use dot mode for presence/status cues when numeric counts are not required.
