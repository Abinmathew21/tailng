Tag overview

Tag is a compact label primitive with optional icon and close action. Use tngTag directly for headless control or <tng-tag> for reusable styled wrappers.

Imports

Choose primitive or component package based on ownership level.

Usage patterns

@for (tag of headlessTags(); track tag.id) { ● {{ tag.label }} }

{{ summary('headless') || 'No tags' }}
@for (tag of plainTags(); track tag.id) { ● {{ tag.label }} }

{{ summary('plain') || 'No tags' }}
@for (tag of tailwindTags(); track tag.id) { ● {{ tag.label }} }

{{ summary('tailwind') || 'No tags' }}

Accessibility baseline

  • Use tngTagClose on a real <button> for keyboard and pointer activation.
  • Set tngTagLabel or label so close controls get clear aria-label text.
  • Keep disabled tags non-removable to avoid inconsistent state transitions.
