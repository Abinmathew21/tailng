Styling contract

CopyButton exposes stable state hooks to style wrapper variants without coupling to internal DOM.

State and variant hooks

Attribute Values Usage
data-appearance outline | solid | ghost Switches wrapper visual treatment.
data-size sm | md Controls control height and horizontal padding.
data-state idle | copying | copied | error Drives transient feedback visuals.

Slot selectors

  • .tng-copy-button__icon for icon sizing/alignment.
  • .tng-copy-button__label for text typography tweaks.
  • .tng-copy-live-region stays visually hidden for announcements.
