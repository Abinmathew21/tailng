Keyboard baseline

  • Tab/Shift+Tab cycles focus inside the dialog panel.
  • Escape closes when dismissal is enabled.
  • Backdrop pointer down closes when closeOnBackdropClick is enabled.
  • Focus returns to the trigger when restoreFocus is active.
