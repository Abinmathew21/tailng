Popover overview

Popover provides anchored floating content with trigger semantics, outside-pointer dismissal, Escape handling, and optional focus restore.

Imports

Use primitives for full markup ownership, or wrapper components for quick defaults.

Keyboard baseline

  • Enter/Space on trigger toggles panel visibility.
  • Escape closes when closeOnEscape is enabled.
  • No focus trap by default; Tab can move outside the panel.
  • Focus can return to trigger when restoreFocus is enabled.
