Tooltip overview

Tooltip provides short helper text on hover or keyboard focus. It stays non-interactive, keeps content concise, and exposes slot/state hooks for custom styling.

Imports

Use primitive directives for full markup ownership, or the wrapper component for quick defaults.

Usage patterns

Headless tooltip with primitive directives and host-managed state.

side: {{ headlessSide() }}

open: {{ plainOpen() ? 'true' : 'false' }}
Tokenized wrapper with utility-first content.

open: {{ tailwindOpen() ? 'true' : 'false' }}

Keyboard baseline

  • Tab to trigger opens tooltip for keyboard users.
  • Shift+Tab or blur closes the tooltip.
  • Escape closes root-managed headless tooltips.
  • Tooltips should stay non-interactive and out of tab order.
