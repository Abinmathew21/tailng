Core packages (recommended)

For most apps, install the Components layer or the primitives layer plus theme, and icons. These are the packages used across the docs and playgrounds.

Minimal install (components + theme)

If you only need styled components and theming, you can add just @tailng-ui/components and @tailng-ui/theme. Primitives and CDK are peer dependencies of many components; add them when you use features that require them.

With icons

To use the TailNG icon set in buttons, menus, and other components, add @tailng-ui/icons.

Peer dependencies

TailNG Components and primitives target modern Angular (v19+). Ensure your workspace uses a compatible Angular version. No additional global styles or polyfills are required beyond your existing Angular and theme setup.
