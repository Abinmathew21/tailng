Overview

tng-timepicker is a time input that implements ControlValueAccessor. It provides a text input (with smart parsing), a clock toggle, and an overlay clock with hour/minute/second navigation. Value is string | null.

Features

12 / 24 Hour Format

Supports 12-hour (hh:mm am/pm) and 24-hour (HH:mm) formats. User can select the preferred format or it can follow the locale.

Seconds Support

Optional : Hidden by default. Shows seconds (ss) only when explicitly enabled.

Slot hooks

slot keys: input, toggle, panel, and many panel sub-elements for full theming.
