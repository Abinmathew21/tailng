Basic usage

Bind with formControlName. Value is string | null.

Validation

Use Validators.required to ensure the user selects a time. Combine it with Validators.pattern to enforce a specific time format.

Example
@if (timeValueCtrl.touched && timeValueCtrl.invalid) {

Please select a valid time.

}

Display format

Supports both: 12-hour format (hh:mm AM/PM) and 24-hour format (HH:mm).

Example
Tip: Use the clock overlay or keyboard (Enter to open, arrows to navigate, Enter to select).
