Styling

The tng-timepicker component uses slot-based micro styling via the slot input. The slot object accepts keys for all internal elements: container, disabled, field, input, toggle, toggleIcon, overlayPanel, panelFrame, panelLayout, title, timeDisplay, hourRail, hourNavPrev, hourNavNext, hourList, hourItem, minuteRail, minuteNavPrev, minuteNavPrev, minuteList, minuteItem, secondRail, secondNavPrev, secondNavNext, secondList, secondItem, period, periodRail, actionBar, cancel, confirm. Each slot targets a specific element in the component structure.

container slot

The root wrapper element. Default: relative.

Example

disabled slot

Wrapper around the field when disabled. Automatically applies opacity-60 and pointer-events-none when disabled.

Example

field slot

Wrapper around the input and toggle button. Default: relative.

Example

input slot

The text input element. Default includes border, padding, focus ring.

Example

toggle slot

The calendar toggle button (right side of the field). Default includes absolute positioning, hover effects.

Example

toggleIcon slot

The calendar icon SVG inside the toggle button. Default: h-5 w-5.

Example

overlayPanel slot

Passed to the internal tng-overlay-panel component.

Example

panelFrame slot

The outer frame of the calendar panel. Default includes border, shadow, rounded corners.
