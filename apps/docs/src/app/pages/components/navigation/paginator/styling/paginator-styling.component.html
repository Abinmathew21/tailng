Styling

tng-paginator uses slot-based micro styling. Slots: root, left, right, button, page, activePage, select, separator.

root slot

Root wrapper. Default: flex flex-wrap items-center justify-between gap-3 text-sm.

Example

left slot

Left section (range text). Default: text-muted-foreground.

Example

right slot

Right section (controls). Default: flex flex-wrap items-center gap-2.

Example

button slot

First/prev/next/last buttons. Default: rounded-md border...

Example

select slot

Page size dropdown. Default: rounded-md border border-border...

Example

separator slot

Ellipsis between page numbers when windowed. Default: px-2 text-muted-foreground.

Example

page / activePage slots

Style page number buttons and the active page. Default: bordered buttons, active = bg-primary.
