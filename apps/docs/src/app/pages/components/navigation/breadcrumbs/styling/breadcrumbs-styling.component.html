Styling

tng-breadcrumbs uses slot-based micro styling. Slots: root, list, item, link, current, disabled, separator.

root slot

Root nav wrapper. Default: flex items-center text-sm text-muted-foreground.

Example

list slot

Ordered list. Default: flex items-center flex-wrap gap-1.

Example

item slot

Each list item. Default: inline-flex items-center.

Example

link slot

Clickable links. Default: text-primary hover:underline.

Example

current slot

Current page (last or explicit). Default: text-foreground font-medium.

Example

disabled slot

Disabled items. Default: opacity-60 pointer-events-none.

Example

separator slot

Separator between items. Default: mx-2 text-slate-400.

Example

Combined slots

Combine link, current, and separator for a custom look.
