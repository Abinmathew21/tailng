Styling

tng-sidenav uses slot-based micro styling. Slots: container, expanded, collapsed, content, footer. Use group and group-data-[collapsed=true]:... on projected content to hide labels when collapsed.

container slot

Root nav base classes. Default: group h-full bg-bg border-r border-border flex flex-col + transition.
