Styling

tng-drawer uses slot-based micro styling. Slots: backdrop, panel, size (left/right), height (top/bottom).

backdrop slot

Backdrop overlay. Default: fixed inset-0 bg-black/40 backdrop-blur-[1px].

panel slot

Panel base classes. Default: bg-bg shadow-xl outline-none.

size slot

Width for left/right placement. Default: w-80.
