Slot-based styling

tng-copy-button supports micro-styling via a single [slot] input. Slot keys are merged with the component's default classes. Pass [slot]="{ container: '…', content: '…' }".

Slot keys

  • container — Button element (variant/size base plus slot classes). Default includes inline-flex items-center gap-1.5 and variant/size classes.
  • content — Content wrapper span (default and slot classes). Default: inline-flex items-center gap-1.5.

Add button-level classes (border-radius, shadow, min-width, etc.) via the container slot.
