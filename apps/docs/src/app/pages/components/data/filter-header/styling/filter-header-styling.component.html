Styling the filter header

The filter header is your custom content inside tngHeader. Style the container (flex/grid, gap, alignment) and the trigger element (icon or button) with Tailwind. Panel overlay styling is controlled via tngFilterTrigger [slot] — see Filter Panel → Styling.

Icon trigger

Use an icon (e.g. funnel) with border and padding. Apply tngFilterTrigger and optional [slot] for the panel.
