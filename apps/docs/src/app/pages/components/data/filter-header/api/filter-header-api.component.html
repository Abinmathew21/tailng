API reference for filter header

tngHeader (template)

Use ng-template tngHeader on a tng-col to define custom header content. Template context: colId, header. Place tngFilterTrigger (and optionally tngSortHeader) inside.

Template context

tngFilterTrigger (directive)

Apply to a clickable element in the header. Opens the filter panel for the column. Pass [slot] to style the panel (see Filter Panel).

Inputs

tng-col [filter]

Set [filter] on tng-col to enable filtering and define the filter type. Required for columns that use tngFilterTrigger.

tngSortHeader (directive)

Optional. Apply to a button (or clickable element) in the header for sortable columns. Pass [colId]. See Sort Header docs for full API.

tng-filter-panel

Add one tng-filter-panel as a child of tng-table. It renders the overlay when a trigger is clicked. For inputs and panel styling, see Filter Panel API.
