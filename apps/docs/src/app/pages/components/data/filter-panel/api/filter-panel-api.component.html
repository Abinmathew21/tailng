API reference for filter-panel

tngFilterTrigger (directive)

Apply tngFilterTrigger to a clickable element (button, icon, etc.) in a column header. It opens the filter panel anchored to that element. Pass [slot] to style the panel.

Inputs

tng-filter-panel (component)

Add tng-filter-panel as a child of tng-table. It provides the overlay that appears when a trigger is clicked. Styling is controlled via the trigger's slot.
