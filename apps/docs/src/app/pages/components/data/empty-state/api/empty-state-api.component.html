API reference for empty state

tng-empty-state (component)

Renders a centered block with optional icon, title, and message. Use inside tables, lists, or cards when there is no data to show.

Inputs

TngEmptyStateSlot

Slot keys for [slot]: container, icon, title, message. Each value is a string of classes merged with the component default. See Styling page for examples.

Table integration

tng-table has an emptyText input that shows a simple message when there are no rows. For a richer empty view, use a custom empty template (if supported) or place tng-empty-state in the same container and show it when rows.length === 0.
