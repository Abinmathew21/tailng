Styling the sort header

The sort header is your custom content inside tngHeader. Apply tngSortHeader on a button (or other clickable element) and style it with Tailwind. Use tng-sort-icon for the default ▲/▼/↕ indicator, or hide it and show your own (e.g. icon font) using the directive's direction in a template.

Button with underline hover

Standard pattern: button with label + tng-sort-icon. Use hover:underline, gap, and alignment to match your table.

Example

Icon-only trigger

Put tngSortHeader on an icon (e.g. sort arrows). The table still toggles sort on click. You can use tng-sort-icon or a custom icon and bind to direction for visual state.
