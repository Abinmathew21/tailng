API reference for sort header

tngSortHeader (directive)

Apply to a clickable element (button or link) inside a tngHeader template. Click toggles sort for that column: asc → desc → none. Sets role="button", tabindex="0", and aria-sort. Keyboard: Enter/Space toggles sort.

Inputs / readonly

tng-sort-icon (component)

Place inside an element that has tngSortHeader. Renders ▲ (asc), ▼ (desc), or ↕ (none). Uses the parent directive's direction. No inputs. Style via CSS (e.g. opacity on the default state).

tng-table sort behavior

tng-table provides sort state and emits when it changes:

  • sortMode — 'client' (default): table sorts rows using column [value]; 'server': you handle data and (sortChange).
  • (sortChange) — emits TngSort ({ active: string, direction: TngSortDir }) when the user toggles sort.

TngSortDir = '' | 'asc' | 'desc'.
