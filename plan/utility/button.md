# Button

Press/action semantics, accessibility, and interaction states.

## Overview

Button component with variants (e.g. primary, secondary, ghost), sizes, and optional icons. Built for accessibility and consistent styling.

## Key points

- Use `tngButton` directive or `TngButtonComponent`; supports `type="button"`, `submit`, etc.
- **Variants** — primary, secondary, outline, ghost, link.
- **Sizes** — sm, md, lg.
- Disabled and loading states; focus ring for keyboard users.
