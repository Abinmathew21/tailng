# Badge

Notification indicator.

## Overview

Small overlay (e.g. dot or count) on an element (e.g. icon or avatar) to indicate notifications or status.

## Key points

- Positioned relative to trigger (corner or edge); count or dot.
- Optional max count (e.g. “99+”); accessible label for screen readers.
- Composes with avatar, icon button, or nav item.
