# CopyButton

Clipboard actions, success/error states, and feedback hooks.

## Overview

Button (or trigger) that copies text to the clipboard and shows success or error feedback (e.g. label change or toast).

## Key points

- Copies a configurable string (or dynamic payload) to clipboard.
- Success/error events and optional duration for reset.
- Often used in code blocks or share flows.
