# Icon

Icon library with Lucide integration.

## Overview

Consistent icon set for UI (buttons, menus, status). Supports Lucide or custom SVGs; sized and colored via CSS or props.

## Key points

- Icon by name or source; size and color from theme or props.
- Accessible: decorative (aria-hidden) or with aria-label when standalone.
- Composes with button, menu, input leading/trailing, badge.
