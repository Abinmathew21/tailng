# Codeblock

Code rendering, highlighting adapters, and line metadata.

## Overview

Displays code snippets with optional syntax highlighting, line numbers, copy button, and configurable language/theme via pluggable adapters (e.g. Shiki).

## Key points

- **Code** — Source string; optional `language` and `theme`.
- **Adapters** — Default or custom highlighter (e.g. plain, Shiki) via `provideTngCodeHighlighting`.
- **UI** — Optional header (title, language badge, copy button), line numbers, caption.
