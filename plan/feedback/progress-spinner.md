# Progress Spinner

Circular loader.

## Overview

Circular indeterminate indicator for loading. Used when progress is unknown (e.g. waiting for response).

## Key points

- Indeterminate by default; optional size and stroke.
- Accessible: role status or progressbar, aria-label (e.g. “Loading”).
- Themed color; optional overlay with message.
