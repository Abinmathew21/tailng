# Skeleton

Loading placeholder blocks.

## Overview

Placeholder shapes (lines, circles, rectangles) that mimic content layout while loading. Reduces perceived wait and layout shift.

## Key points

- Configurable shapes and count; optional animation (pulse/shimmer).
- No semantic content; decorative or aria-hidden when used as placeholder.
- Composes with card, list, or text layout.
