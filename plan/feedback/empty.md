# Empty

Empty-state layout.

## Overview

Placeholder when there is no data or no results (e.g. empty list or search). Often includes illustration, message, and optional action (e.g. “Add item”).

## Key points

- Slot or template for message and optional CTA.
- Themed to match app; accessible text for screen readers.
- Composes with list or table container.
