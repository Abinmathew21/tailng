# Progress Bar

Linear progress indicators.

## Overview

Horizontal bar showing progress (0–100% or indeterminate). Used for uploads, steps, or loading states.

## Key points

- Value (number or indeterminate); optional label and min/max.
- Accessible: role progressbar, aria-valuenow/valuemin/valuemax (or aria-valuetext for indeterminate).
- Themed color and optional stripe animation.
