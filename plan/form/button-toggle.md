# Button Toggle

Toggleable button options.

## Overview

Button or button-like control that toggles between pressed and unpressed. Can be used alone or in a group (see Toggle Group).

## Key points

- Pressed state reflected visually and in accessibility (aria-pressed).
- Optional icon or label for on/off; can be used in toolbar.
- Composes with menu or toolbar for tool options.
