# Chips

Tokenized input and removable item chip usage patterns.

## Overview

Chips represent selected items or tags that can be removed (e.g. in multi-autocomplete or filter UIs).

## Key points

- Each chip is a compact token with optional remove action.
- Often used with multi-select or tag input.
- Accessible remove control (button or icon) per chip.
