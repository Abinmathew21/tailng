# Input OTP

Segmented OTP entry.

## Overview

Multi-field input for one-time codes (e.g. verification digits). Each character or digit has its own slot; focus moves automatically between segments.

## Key points

- Number of segments configurable (e.g. 4, 6).
- Paste splits across segments; backspace moves to previous when empty.
- Optional mask or type (numeric/alphanumeric).
