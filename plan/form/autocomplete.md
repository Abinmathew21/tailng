# AutoComplete

Single-value autocomplete patterns and filtering behavior.

## Overview

Combines a text input with a listbox (or dropdown) to let users type and select a single value from filtered options.

## Key points

- Input drives filtering; listbox shows matching options.
- Single selection; value typically reflected in the input.
- Keyboard and mouse selection with accessible focus management.
