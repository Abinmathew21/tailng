# MultiAutocomplete

Chip-based autocomplete selection for multiple values.

## Overview

Autocomplete that allows multiple selections, often displayed as chips or tags, with add/remove and inline filtering.

## Key points

- Multiple selected values shown as chips (or similar).
- New selections added from filtered list; chips can be removed.
- Composes listbox with input and chip/tag UI.
