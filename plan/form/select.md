# Select

Single-choice dropdown.

## Overview

Dropdown that allows choosing one option from a list. Trigger shows current selection; opening shows the full list.

## Key points

- Trigger (button or field) opens listbox; one value selected.
- Keyboard: Enter/Space open; arrows move; Enter selects.
- Composes with listbox; supports custom option rendering and search.
