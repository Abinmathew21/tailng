# Combobox

Text input with options popup.

## Overview

Input that opens a list of options (e.g. suggestions or choices). User can type to filter and select; selection can update the input value.

## Key points

- Input + listbox/popover; filtering and single selection.
- Keyboard: type to filter, arrows to move, Enter to select.
- Composes with listbox and popover; can be editable or select-only.
