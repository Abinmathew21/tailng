# Checkbox

Checked/unchecked/mixed tri-state.

## Overview

Checkbox for boolean or tri-state (checked, unchecked, indeterminate). Used for single options or as part of a multi-select list.

## Key points

- Two-state (on/off) or tri-state (indeterminate for “some selected”).
- Keyboard: Space toggles; optional group behavior.
- Composes with label; accessible name and state announced.
