# Toggle Group

Grouped toggle controls.

## Overview

A set of toggle buttons where one or more can be selected (single or multiple mode). Used for options like alignment, view mode, or filters.

## Key points

- Single-select (radio-like) or multi-select (checkbox-like).
- Keyboard navigation and selection; optional disabled items.
- Composes with toggle/button primitive.
