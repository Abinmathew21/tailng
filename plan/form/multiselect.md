# MultiSelect

Multiple-selection dropdown behavior and state handling.

## Overview

Dropdown or listbox that allows selecting multiple options, with state reflected in the trigger (e.g. count or chip list).

## Key points

- Multiple selected values; toggle on option click.
- Checkbox or highlight state per option.
- Clear-all and keyboard support for multi-select.
