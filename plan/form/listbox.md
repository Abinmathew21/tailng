# ListBox

Listbox interaction patterns and keyboard behavior reference.

## Overview

Headless listbox primitive for single or multiple selection with full keyboard support (arrow keys, typeahead, Home/End) and ARIA semantics.

## Key points

- Use `tngListbox` on a container and `tngOption` with `tngValue` on each option.
- Supports single value, multi-select, typeahead, and focus loop.
- Composes with dropdowns, autocomplete, and custom list UIs.
