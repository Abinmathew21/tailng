# Charts

Country metrics with bar/line chart.

## Overview

Chart components for data visualization (e.g. bar, line). Playground example shows country metrics; can be extended for other chart types.

## Key points

- Data-driven; configurable axes, series, and tooltips.
- Accessible: optional table or description for data; keyboard/screen reader support where applicable.
- Composes with theme and optional legend.
