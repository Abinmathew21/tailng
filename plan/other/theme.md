# Theme

Switch presets/modes and inspect semantic tokens.

## Overview

Theme system for presets (e.g. default, minimal), light/dark mode, and semantic tokens (color, spacing, typography). Used in playground to switch and inspect.

## Key points

- Presets and mode (light/dark); tokens applied as CSS custom properties.
- Inspect or document tokens for design-system alignment.
- Composes with app config (provideTailngTheme) and component contracts.
