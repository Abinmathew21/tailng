# Navigation Menu

Structured navigation list.

## Overview

Navigation component for hierarchical or flat link lists, often used as main app or section nav. Supports nested items and active state.

## Key points

- List of links or submenus; can be vertical or horizontal.
- Active state and ARIA for current page/section.
- Composes with menu or list patterns for expandable sections.
