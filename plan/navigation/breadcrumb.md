# Breadcrumb

Hierarchical navigation trail.

## Overview

Shows the path from root to current page (e.g. Home > Section > Page). Each segment can be a link except the current one.

## Key points

- List of segments; last segment is typically current (non-link).
- Optional separator between items (e.g. `/`, `›`).
- Accessible list/nav semantics and optional collapse on narrow viewports.
