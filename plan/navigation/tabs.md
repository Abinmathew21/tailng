# Tabs

Tabbed container surface.

## Overview

Switches between panels via tab triggers. One panel visible at a time; selection is keyboard- and pointer-accessible.

## Key points

- Tab list (horizontal or vertical) and matching panels; value binds selection.
- Keyboard: arrows move focus/selection, Enter/Space activate.
- Optional scroll buttons, lazy panel mounting, and URL sync.
