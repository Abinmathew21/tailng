# Menu

Contextual menu interactions, keyboard flow, and submenu behavior.

## Overview

Dropdown or popover menu with menu items, optional submenus, and full keyboard support (arrows, Enter, Escape).

## Key points

- Trigger opens menu; items can be actions or submenu triggers.
- **Keyboard** — Arrow keys, Enter to activate, Escape to close.
- Composes with menubar, dropdown, or context trigger.
