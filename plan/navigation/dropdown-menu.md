# Dropdown Menu

Triggered menu panel.

## Overview

A menu that opens from a trigger (button or link). Combines trigger semantics with a menu panel for actions or navigation.

## Key points

- Trigger opens/closes the menu on click or keyboard.
- Menu panel positions relative to trigger (e.g. below, aligned).
- Keyboard navigation within menu; Escape or outside click closes.
