# Menubar

Top-level menubar patterns with owned menu integration.

## Overview

Horizontal menubar (e.g. app menu) where each item can open a dropdown menu. Keyboard navigation and ARIA menubar/menu semantics.

## Key points

- Menubar contains triggers; each trigger opens one menu.
- Arrow keys move between items and open/close menus.
- Composes with Menu component for dropdown content.
