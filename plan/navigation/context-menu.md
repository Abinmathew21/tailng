# Context Menu

Context-triggered actions.

## Overview

Menu that opens on context (e.g. right-click) at the pointer position. Used for contextual actions on a target element or region.

## Key points

- Opens on contextmenu event; position follows pointer.
- Same menu semantics as dropdown menu (keyboard, dismiss).
- Often used in lists, tables, or canvas UIs.
