# Tooltip

Hover/focus helper text.

## Overview

Short text that appears on hover or focus (e.g. describing an icon button). Dismisses when pointer leaves or focus moves.

## Key points

- Trigger (element) and content (string or template); delay and placement configurable.
- Accessible: can be referenced via aria-describedby; keyboard focus shows tooltip.
- Avoid large content; use popover for richer content.
