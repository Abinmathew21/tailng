# Dialog

Backdrop, focus-trap.

## Overview

Modal dialog with backdrop and focus trapped inside. Used for confirmations, forms, or critical content that requires user response.

## Key points

- Backdrop dims background; focus stays in dialog; Escape closes (if dismissible).
- Optional title, description, and actions (e.g. Cancel/Confirm).
- Accessible role dialog; aria-modal and focus restoration on close.
