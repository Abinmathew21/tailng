# Plain CSS Setup

Set up TailNG Components with semantic tokens using vanilla CSS.

## Overview

Use TailNG without Tailwind: wire semantic design tokens and component contracts via plain CSS (or CSS custom properties).

## Key points

- Load theme and component contract styles (e.g. from `@tailng-ui/theme`).
- Apply tokens for colors, spacing, and typography.
- No build-time utility layer required.
