# Quick Start

Fast path to scaffold and run your first TailNG component page.

## Overview

Get TailNG Components running in your Angular app in a few minutes: install packages, register the theme, and render your first component.

## Key steps

- **Install** — Add `@tailng-ui/components`, `@tailng-ui/primitives`, `@tailng-ui/cdk`, `@tailng-ui/theme`, and `@tailng-ui/icons` (pnpm, npm, or yarn).
- **Theme** — Call `provideTailngTheme()` in your app config providers.
- **Use** — Import standalone components (e.g. `TngButtonComponent`) and use them in templates.
