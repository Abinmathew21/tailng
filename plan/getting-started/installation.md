# Installation

Package-level install matrix for TailNG Components and peer layers.

## Overview

Install TailNG at the level that fits your project: full set (components, primitives, CDK, theme, icons), minimal (components + theme), or with icons.

## Options

- **Core (recommended)** — Components, primitives, CDK, theme, icons. Supports pnpm, npm, yarn.
- **Minimal** — `@tailng-ui/components` and `@tailng-ui/theme` only.
- **With icons** — Full set including `@tailng-ui/icons`.

## Peer dependencies

- Modern Angular (v19+). No extra global styles or polyfills required beyond your theme setup.
