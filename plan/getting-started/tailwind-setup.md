# Tailwind Setup

Integrate TailNG Components with utility-first styling in Tailwind.

## Overview

Use TailNG with Tailwind CSS: theme tokens and component styles work alongside utility classes for layout and appearance.

## Key points

- Configure Tailwind to include TailNG theme/content if needed.
- Use semantic tokens (e.g. `var(--tng-semantic-*)`) in Tailwind or component CSS.
- Combine TailNG components with Tailwind utilities in templates.
