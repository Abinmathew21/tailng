# Stepper

Multi-step flow surface.

## Overview

Shows steps in a linear flow (e.g. wizard or checkout). Steps can be clickable for navigation; current step and completion state are visible.

## Key points

- Linear or horizontal layout; optional step labels and connectors.
- Current step, completed, optional; keyboard and pointer navigation.
- Composes with form sections or route-based steps.
