# Card

Validate card primitives.

## Overview

Container component for grouped content: header, body, footer, and optional media. Used for content blocks, list items, or dashboards.

## Key points

- Slots or sections for header, content, footer; optional image/media.
- Variants: outline, filled, elevated; optional padding and interactive state.
- Composes with button, link, or list inside.
