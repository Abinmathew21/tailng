# Accordion

Expand/collapse panels.

## Overview

Vertical list of headers that expand to show content. One or multiple panels can be open (single vs multiple mode); keyboard navigable.

## Key points

- Each item has trigger and panel; trigger toggles panel visibility.
- Optional single-expand (collapse others) or multi-expand.
- Arrow keys move focus; Enter/Space toggle.
