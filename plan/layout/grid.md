# Grid

Keyboard-navigable grid.

## Overview

Two-dimensional grid of cells (e.g. calendar, data grid, or picker). Arrow keys move between cells; selection and activation are configurable.

## Key points

- Rows and columns; optional row/col span.
- Keyboard: arrows, Home/End, Page Up/Down; Enter/Space activate.
- Composes with listbox or custom cell content.
