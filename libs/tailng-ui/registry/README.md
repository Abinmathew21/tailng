# @tailng-ui/registry

Copy-paste registry source for TailNG CLI (`tailng add ...`).

## Registry Items

Current copy-paste items:

- `button`
- `card`
- `checkbox`
- `dialog`
- `input`
- `popover`
- `radio`
- `textarea`

Use exported helpers to list and resolve registry items.
